# Create: Chemical Response

> **⚠️ This project is currently in very early development.**

Create: Chemical Response (CCR) is an upcoming addon for the Create mod focused on chemical response, hazardous material handling, emergency equipment, and industrial cleanup.
NOTE: 
This mod does not add any chemicals. In the future Create: Big Cannons and CBC: Chemical warfare will be required to add the chemical hazards. Other compatibility may come in the future with mods like destroy.
Also this mod is mainly supposed to be a catalyst for roleplay scenerios, not a chemical disaster simulator.
## Features

> *Coming soon.*

Planned features include:
- Chemical response equipment
- Hazmat-themed blocks and decorations
- Emergency tools
- Cleanup mechanics
- And more...

## Requirements

- Minecraft 1.20.1
- Minecraft Forge 47.x
- Create 6.0.x
- One day 1.21.1 support may be added. support for create: fly is not planned.
## Installation

1. Install Minecraft Forge for Minecraft 1.20.1.
2. Install Create.
3. Download the latest release of Create: Chemical Response.
4. Place both mods into your `mods` folder.
5. Launch Minecraft.
